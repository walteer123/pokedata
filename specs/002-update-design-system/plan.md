# Implementation Plan: Update Design System to Modern Material 3

## Summary

Update the Pokedex app's design system to the latest Material 3 (Material You) guidelines. This includes: modernizing the color palette with full M3 tonal schemes, adding custom shape tokens, integrating Google Fonts (Inter + Roboto), fixing the broken Pokemon type badges to display correct type colors, creating shared modern UI components (TypeBadge, PokemonCard, ModernBottomNav), adding a bottom navigation bar for primary navigation, and implementing smooth screen transition animations. All changes are purely presentational — no data models, APIs, or business logic are affected.

## Technical Context

**Language/Version**: Kotlin 2.0.21  
**Primary Dependencies**: Jetpack Compose (BOM 2025.03.01), Material 3, Koin 3.5.6, Navigation Compose 2.9.0, Coil 2.7.0, Paging 3  
**New Dependencies**: Bundled TTF fonts (Inter, Roboto) in `res/font/`  
**Storage**: No changes — Room database unchanged  
**Testing**: Compose UI tests for new shared components, visual verification for existing screens  
**Target Platform**: Android (minSdk 26, targetSdk 35)  
**Project Type**: android-app (modular, 9 modules + build-logic)  
**Performance Goals**: 60fps transitions (300ms max), no dropped frames, cold start < 3s maintained  
**Constraints**: Offline-first (fonts bundled, not fetched), WCAG AA contrast (4.5:1 minimum), mid-tier device support  
**Scale/Scope**: Design system update affecting `core:designsystem` module and all 4 feature screens

### Key Technical Decisions

1. **Fonts**: Bundled TTF files (not Google Fonts API) for offline availability
2. **Shapes**: Custom `Shape.kt` with M3 token scale (4dp–28dp rounded corners)
3. **Type Badges**: Direct type color application with luminance-based text contrast
4. **Navigation**: Bottom `NavigationBar` replacing TopAppBar icon navigation for primary destinations
5. **Transitions**: Navigation Compose `slideIntoContainer`/`slideOutOfContainer` APIs
6. **Colors**: Updated M3 tonal palettes preserving Pokemon-themed primary red
